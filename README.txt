# 💡 AI Project Recommender

Une application web intelligente conçue pour aider les développeurs à identifier les meilleures idées de scripts à vendre sur CodeCanyon, en fonction :
- des tendances actuelles du marché
- des compétences du développeur
- et des projets les plus simples ou les plus rentables.

---

## 🚀 Fonctionnalités

- 📊 Recommandation d’idées de projets tendance
- 🎯 Suggestions personnalisées basées sur les compétences de l’utilisateur
- 🌍 Interface web responsive et intuitive
- 🧠 Intelligence "guidée" simple mais efficace
- 📁 Fichier JSON pour mise à jour manuelle des tendances
